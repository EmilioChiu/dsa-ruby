LRU Cache Reference

Estructura

  • Hash Map: Para acceso O(1) a los nodos.
  • Doubly Linked List: Para reordenar nodos en O(1).
  • Dummy Nodes: head y tail fijos.

Operaciones Clave

  • GET(key): Si existe, remove(node) + insert(node).
  • PUT(key, val): insert(node). Si excede capacidad, remove(tail.prev).

Diagrama de Punteros (Inserción MRU)

Para insertar un nodo N entre head y A:

1. nxt = head.next
2. head.next = N
3. nxt.prev = N
4. N.prev = head
5. N.next = nxt

Resultado: head ⇄ N ⇄ A

Referencia: Queues (Colas)

Una Queue es una estructura lineal que sigue el principio FIFO (First-In, First-Out).

Operaciones Principales

Operación Descripción Complejidad (Ideal)
enqueue(x) Añade un elemento al final de la cola. O(1)
dequeue() Elimina y retorna el elemento al frente. O(1)
peek() / front() Retorna el elemento al frente sin eliminarlo. O(1)
isEmpty() Verifica si la cola no tiene elementos. O(1)

Implementación en Ruby

En Ruby, un Array puede actuar como Queue, pero shift es O(n):


queue = []
queue << 1          # Enqueue
queue << 2
front = queue.shift # Dequeue (Retorna 1, O(n))

Patrones Comunes

  • Two Stacks: Simular FIFO usando dos estructuras LIFO.
  • Circular Buffer: Optimizar espacio usando un array de tamaño fijo y punteros head y tail.
  • BFS: Uso fundamental para explorar nodos por nivel.

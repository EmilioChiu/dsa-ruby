BS en Rotated Array

Tres patrones: Search, Find Min, Peak Element

1. Search in Rotated Sorted Array (LC 33)

Problema: Encontrar un target en un array rotado. Medium

Clave: Determina cual mitad esta ordenada, luego verifica si el target esta ahi.

def search(nums, target)
  left, right = 0, nums.length - 1

  while left <= right
    mid = left + (right - left) / 2
    return mid if nums[mid] == target

    if nums[left] <= nums[mid]         # izq ordenada
      if nums[left] <= target && target < nums[mid]
        right = mid - 1
      else
        left  = mid + 1
      end
    else                                   # der ordenada
      if nums[mid] < target && target <= nums[right]
        left  = mid + 1
      else
        right = mid - 1
      end
    end
  end
  -1
end

2. Find Minimum in Rotated Array (LC 153)

Problema: Encuentra el elemento minimo en un array rotado. Medium

Clave: Compara nums[mid] con nums[right]. Si mid > right, el minimo esta a la derecha.

def find_min(nums)
  left, right = 0, nums.length - 1

  while left < right
    mid = left + (right - left) / 2

    if nums[mid] > nums[right]
      left  = mid + 1          # minimo a la derecha
    else
      right = mid              # minimo a la izquierda (incl. mid)
    end
  end
  nums[left]
end

3. Find Peak Element (LC 162)

Problema: Encuentra cualquier pico (elemento mayor que sus vecinos). Medium

Clave: Sigue la pendiente ascendente. Compara nums[mid] con nums[mid+1].

def find_peak_element(nums)
  left, right = 0, nums.length - 1

  while left < right
    mid = left + (right - left) / 2

    if nums[mid] < nums[mid + 1]
      left  = mid + 1           # pico a la derecha
    else
      right = mid               # pico a la izquierda (incl. mid)
    end
  end
  left
end

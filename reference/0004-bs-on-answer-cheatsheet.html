BS en Respuesta

Busqueda binaria sobre el espacio de soluciones posibles

🔎 Cuando Usarlo

Las 3 senales:

  1. Pide un minimo o maximo valor que cumple algo
  2. Puedes escribir feasible?(candidato) que retorna true/false
  3. feasible? es monotonica (F...F...T...T o viceversa)

📈 Monotonicidad

La condicion debe ser monotona en una direccion:

# Buscando MINIMO factible:
# F, F, F, F, T, T, T, T
#          ^ primer true = respuesta

# Buscando MAXIMO factible:
# T, T, T, T, F, F, F, F
#          ^ ultimo true = respuesta

🛠 Template: Minimo Factible

def find_min(problem)
  left  = MIN   # menor valor posible
  right = MAX   # mayor valor posible

  while left < right
    mid = left + (right - left) / 2

    if feasible?(mid)
      right = mid   # funciona, busca menor
    else
      left = mid + 1 # no funciona, busca mayor
    end
  end

  left
end

🛠 Template: Maximo Factible

def find_max(problem)
  left  = MIN
  right = MAX

  while left < right
    mid = left + (right - left + 1) / 2  # mid "alto"

    if feasible?(mid)
      left = mid      # funciona, busca mayor
    else
      right = mid - 1 # no funciona, busca menor
    end
  end

  left
end

Nota el + 1 en el calculo de mid — evita loop infinito cuando left y right son consecutivos.

⌨ Ceiling Division

Calcular horas sin usar floats:

hours = (pile + k - 1) / k

Equivalente a (pile / k.to_f).ceil pero sin punto flotante.

⚠ Errores Comunes

  • Usar left <= right en vez de < — loop infinito
  • Olvidar el + 1 en el template de maximo
  • Rango incorrecto (left muy pequeno, right muy grande)
  • feasible? no monotónica
  • No reconocer si buscas min o max

📚 Checklist para Resolver

  1. Identificar que pide el problema (minimo o maximo?)
  2. Definir el rango: left = valor minimo valido, right = valor maximo posible
  3. Escribir feasible?(mid) — simula si con ese valor se cumple la condicion
  4. Verificar monotonicidad: si mid funciona, todos los mayores tambien? (o menores)
  5. Elegir template (minimo o maximo factible)
  6. Probar con ejemplos
Si no puedes escribir feasible? claramente, no es BS en respuesta.

# Mission: Estructuras de Datos y Algoritmos en Ruby

## Why
Dominar los fundamentos de las estructuras de datos y algoritmos (DSA) utilizando Ruby para superar entrevistas técnicas de código en empresas de tecnología y consolidar habilidades de diseño de software eficiente.

## Success looks like
- Implementar estructuras de datos fundamentales (listas enlazadas, pilas, colas, árboles, grafos) desde cero en Ruby.
- Analizar con precisión la complejidad temporal y espacial en notación Big-O de cualquier fragmento de código.
- Resolver problemas de nivel Easy y Medium de LeetCode correspondientes a los patrones esenciales (NeetCode 150) con código Ruby limpio e idiomatico.
- Explicar verbalmente y con claridad las decisiones de diseño y trade-offs entre diferentes soluciones a un entrevistador.

## Constraints
- **Lenguaje**: Únicamente Ruby.
- **Régimen**: Aprendizaje autodidacta estructurado en 12 semanas (~180 horas totales).
- **Enfoque práctico**: Cada concepto teórico debe estar respaldado por ejercicios de código ejecutables con pruebas.

## Out of scope
- Desarrollo web frontend o backend tradicional (Rails, Sinatra, etc.).
- DevOps, bases de datos o infraestructura.
- Problemas de competitividad avanzada que excedan el alcance de entrevistas técnicas estándar (ej. segment trees, network flow).
